Welcome to BSDC — Bangladesh Software Development Community

The ultimate hub for every Bangladeshi developer, programmer, and tech enthusiast — from Dhaka to Chittagong, from student to CTO.

What is BSDC? | BSDC কী?

BSDC (Bangladesh Software Development Community) is an open, free, community-driven platform built specifically for software developers in Bangladesh. Whether you write code in Python, JavaScript, PHP, Java, Flutter, Dart, Go, Rust, or any other language — BSDC is your home.

BSDC হলো বাংলাদেশের সফটওয়্যার ডেভেলপারদের জন্য একটি মুক্ত, বিনামূল্যে, কমিউনিটি-চালিত প্ল্যাটফর্ম। আপনি যদি Python, JavaScript, PHP, Flutter বা যেকোনো প্রযুক্তিতে কাজ করেন — BSDC আপনার জন্য।

Ask & Answer Questions | প্রশ্ন করুন ও উত্তর দিন

Stuck on a bug? Need architectural advice? BSDC's Q&A system works like Stack Overflow but with a Bangladeshi developer context. Post your question, get answers from experienced Bangladeshi engineers, mark solutions as solved, and upvote the best answers.

Share Code Snippets | কোড শেয়ার করুন

BSDC supports full syntax highlighting for 50+ programming languages. Share reusable code snippets, utility functions, algorithms, and complete project structures. Snippet pages are individually SEO-indexed so your code reaches developers searching Google.

Write Wikis & Blogs | উইকি ও ব্লগ লিখুন

Have deep expertise in Django REST Framework, or want to explain how BGP routing works in Bangla? Write a comprehensive Wiki or Blog post on BSDC. Your content gets its own dedicated page, fully indexed by Google, Bing, and AI systems like ChatGPT and Gemini.

Share Your Projects | প্রজেক্ট শেয়ার করুন

Built something amazing? Share your open-source or commercial projects with the Bangladeshi developer community. Get feedback, find collaborators, and showcase your work. BSDC project pages include GitHub integration, live demo links, and full tech stack documentation.
